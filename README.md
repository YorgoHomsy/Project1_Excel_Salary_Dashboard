# Project 1 | Excel Salary Dashboard

## Introduction
This interactive Excel dashboard helps job seekers explore salaries for different data-related roles and understand how **location, job type, and experience** affect compensation.  
The dataset comes from my Excel course and includes job titles, salaries, locations, and key skills.
